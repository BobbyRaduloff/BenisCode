# BENIS ENCODING

## ENCODING METHOD
This program encodes messages by placing single bits in the capitalization of the word "benis". This is best shown by example:
```
bEnIS
01011 = 11
```

## ENCODING TABLE
| Number | Corresponding Character |
| ------ | ----------------------- |
| 0 | SPACE | 
| 1 - 26 | lowerCaseAlphabet |
